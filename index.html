    Como surgiu a Internet

    Tudo começou com o mundo entendendo que a comunicação era a chave, para qualquer tipo de progresso. Então começaram a buscar maneiras de se comunicar entre pontos distantes. O primeiro equipamento utilizado com esse intuito que obteve grande exito foi o telegrafo, já que conseguia operar em grandes distancias inclusive dentro de navios.

    Tempos depois foi criada uma empresa com intuito de desenvolver um sistema mais eficaz para conectar pontos distantes seu nome Darpa Defense Advance Research Projects Agency. Dentro desta agencia foi desenvolvido o que seria o primeiro prototipo da internet chamada de Arpanet, sistema de comunicação em pacotes, enviados um a um e que permitia múltiplos destinos. A primeira conexão estabeleciada foi feita em 1969. Tempos depois em 1989 e depois de muita pesquisa dentro da área Tim Berners - Lee uniu TCP/IP e hipertexto e criou World Wide Web (www). E em 1991 foi desenvolvida a primeira página web http://info.cern.ch/.

    O que são Clients

    O conceiro client é de onde o usuario está enviando uma requisição, no caso pode ser do computador, notebook, smarthphone. Você envia uma requisição ao servidor ele retorna o html, css, js, imagens... e utiliza o navegador para armazenar os caches no seu computador tudo isso utilizando a internet.

    Conceito de Servers

    Servers são o outro lado é quem recebe a requisição do usuário e retorna o html, js, css, imagens entre outros já que nos servers é onde estão armazenados os arquivos. Existem também servidores locais tipo LAN que por estarem in loco não utilizam a internet e armazenam informações de vários computadores que estão no mesmo local.

    Tipos de Servidores

    1. Servidor Proxy cria algumas regras de acesso a internet, por exemplo bloqueia acesso aa alguns sites.
    2. Servidor Firewall protege com relação a acesso externo.
    3. Servidor Web, servidor dos navegadores.
    4. Servidor e-mail, Armazena e-mails.
    5. Servidor Database, Aplicação instalada no servidor para acessar o banco de dados.
    6. Servidor DNS, redireciona um ip responsavel por fazer com que o dominio seja localizado na internet.

    Hospedagem de Sites

    Você coloca seus arquivos que constituem o site dentro do servidor para os clientes acessarem. Conexão FTP o provedor de hospedagem fornece os dados de acesso FTP ao servidor.

    Ferramentas Utilizadas (Visual Studio Code), (Extensão Live Server), (Extensão Emmet). Utilizando essas ferramentas você vai conseguir criar seus códigos e acompanhar todas as aulas colocando toda teoria em prática. Inspetor de elementos quando você clica com o botão direito e seleciona a opção inspecionar ou utilizado o f12 na web página muitos importante para você ter acesso aos arquivos html, css e js.
